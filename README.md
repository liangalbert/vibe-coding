# Vibe Coding

AI-assisted coding notes, workflows, experiments, and project retrospectives.

## About

This space is for sharing how I build software with AI as a thinking partner:
from quick prototypes to reusable workflows, from prompts to debugging notes,
from small demos to complete project writeups.

- Tool comparisons and practical tips

### Projects

- Mini apps and prototypes
- Frontend experiments
- Automation scripts
- Research-adjacent tools

### Notes

- Lessons learned
- Useful commands
- Common mistakes
- Ideas worth revisiting
